## 🎯 다음 단계

사용자가 실제로 기능을 테스트하려면:

1. `.env.local` 파일에 Google API Key 설정 필요
2. 대시보드에서 주제 입력 후 "기사 생성" 버튼 클릭
3. 실시간 스트리밍 응답 확인
4. 멀티 포맷 변환 버튼 테스트
5. 설정 탭에서 시스템 프롬프트 커스터마이징

## 📝 참고사항

- API Key가 설정되지 않은 경우, 에러 메시지가 표시됩니다
