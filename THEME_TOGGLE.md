# 테마 전환 기능 추가

## ✅ 구현 완료

웹사이트에 다크 모드와 라이트 모드를 전환할 수 있는 버튼을 추가했습니다.

### 주요 기능

**테마 전환 버튼**은 화면 우측 상단에 고정되어 있으며, 다음과 같은 특징을 가지고 있습니다:

- **위치**: 화면 우측 상단 (`fixed top-4 right-4`)
- **디자인**: 원형 버튼 (`rounded-full w-12 h-12`)
- **아이콘**: 
  - 다크 모드: Sun 아이콘 (노란색)
  - 라이트 모드: Moon 아이콘 (회색)
- **저장**: LocalStorage에 테마 설정 저장

### 테마별 스타일

**다크 모드** (기본값)는 다음과 같은 스타일을 사용합니다:
- 배경: `bg-slate-950` (매우 어두운 남색)
- 텍스트: `text-slate-100` (밝은 회색)
- 카드: `bg-white/5 backdrop-blur-lg border-white/10` (Glassmorphism)
- 입력창: `bg-white/5 border-white/20`
- 설명: `text-slate-400`

**라이트 모드**는 다음과 같은 스타일을 사용합니다:
- 배경: `bg-gradient-to-br from-slate-50 to-blue-50` (밝은 그라데이션)
- 텍스트: `text-slate-900` (어두운 회색)
- 카드: `bg-white/80 backdrop-blur-lg border-slate-200` (밝은 Glassmorphism)
- 입력창: `bg-white border-slate-300`
- 설명: `text-slate-600`

### 사용 방법

1. 화면 우측 상단의 **Sun/Moon 아이콘** 클릭
2. 테마가 즉시 전환되며 토스트 메시지 표시
3. 선택한 테마는 LocalStorage에 저장되어 다음 방문 시에도 유지

### 개선 사항

라이트 모드에서는 배경이 밝아져 모든 텍스트와 UI 요소가 명확하게 보입니다. 설정 페이지의 가독성이 크게 향상되었으며, 사용자가 선호하는 테마를 자유롭게 선택할 수 있습니다.

**다크 모드**는 눈의 피로를 줄이고 미래적인 느낌을 주며, **라이트 모드**는 밝은 환경에서 더 나은 가독성을 제공합니다.
